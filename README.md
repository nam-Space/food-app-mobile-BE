### Các bước cần làm để chạy dự án Backend Shop App Mobile

#### 1. Cài đặt thư viện với câu lệnh: npm install --omit=dev

#### 2. Update file .env (nếu cần thiết)

#### 3. Chạy dự án với câu lệnh: npm start
